Detailed Description

Represents a request with input and output IP addresses and processing time.

The Request struct represents a request with input and output IP addresses and a processing time. It generates random IP addresses within the range of 0.0.0.0 to 255.255.255.255 for both the input and output IP addresses. The processing time is an integer value.
